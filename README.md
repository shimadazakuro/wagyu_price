# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

* リモートリポジトリが、ネット上に配置して複数人で共有するためのリポジトリ。
  ローカルリポジトリが、開発者一人ひとりが使用するために自分のPC上に配置するためのリポジトリ。

## プッシュとマージの違いは何でしょうか？

* プッシュが、変更内容をリモートの同名ブランチに反映させ、
　マージが、ブランチに変更履歴を反映させるため、
　リモートリポジトリに反映させるものが違う。

## コミットとプッシュの違い

* コミットが、ローカルのブランチに変更内容を反映させ、
　プッシュが、リモートのブランチに変更内容を反映させるため、
　反映させるための、リポジトリが違う。

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

* どんな編集を行なったのかを示し、分かりやすく書く。
　ただ、プロジェクトによってはルールが存在するため、その時は確認する。

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

* 変更内容をマージをしてくださいという依頼を行い、
　事前にバグの発生を抑えるために他の人からのコードレビューをしてもらう事行うか行わないかの違い。

## コンフリクトを起こしてしまった場合、どう対処すべきですか？

* 1.先にマージされた内容を取り込む
  2.後にマージされた内容を取り込む
  3.どちらの変更内容も取り込む
